# 基金实时估值查询

一个使用纯原生 HTML、CSS3、JavaScript 编写的基金实时估值查询页面，兼容 PC Web 和 H5 页面展示，支持单基金查询、最近查看缓存、基金分组管理和分组明细页批量查看。

## 功能说明

- 支持输入 6 位基金代码查询实时估值
- 展示基金名称、基金代码、实时净值、前一日净值、净值日期、估值时间
- 根据涨跌幅自动切换展示颜色
- 支持缓存最近查看的 10 条基金记录
- 支持创建本地基金分组
- 支持进入分组明细页查看整组基金数据
- 单个分组最多保存 10 只基金
- 分组明细页会按顺序逐笔请求基金估值，并在每笔请求之间自动延迟，降低瞬时并发压力
- 支持在分组明细页新增和删除基金代码
- 所有分组和历史记录都仅保存在浏览器本地 `localStorage`

## 页面结构

- `index.html`：主页面，负责单基金查询、最近查看记录、分组创建和分组入口展示
- `group-detail.html`：分组明细页面，负责查看整组基金数据、添加基金代码、删除基金代码

## 项目文件

- `index.html`：主页面
- `group-detail.html`：分组明细页
- `assets/app.css`：公共样式
- `assets/fund-common.js`：公共数据处理、JSONP 请求、分组和历史缓存逻辑
- `nginx.conf`：静态部署时可参考的 Nginx 配置

## 接口说明

页面使用东方财富基金估值 JSONP 接口：

`http://fundgz.1234567.com.cn/js/基金代码.js?rt=时间戳`

示例返回：

```js
jsonpgz({
  "fundcode": "017470",
  "name": "嘉实上证科创板芯片ETF发起联接C",
  "jzrq": "2026-04-30",
  "dwjz": "2.5037",
  "gsz": "2.7068",
  "gszzl": "8.11",
  "gztime": "2026-05-06 13:39"
});
```

字段说明：

- `fundcode`：基金代码
- `name`：基金名称
- `jzrq`：净值日期
- `dwjz`：前一交易日净值
- `gsz`：实时估值净值
- `gszzl`：实时涨跌幅百分比
- `gztime`：估值时间

## 使用方式

### 本地打开

直接使用浏览器打开 `index.html` 即可。

### 页面操作

1. 在主页面输入基金代码可查询单只基金信息
2. 在主页面创建分组后，可点击分组卡片进入分组明细页
3. 在分组明细页可新增或删除基金代码
4. 分组页进入后会自动顺序加载组内全部基金数据

### 静态部署

将项目部署到任意静态服务器即可，例如 Nginx。

## 本地缓存逻辑

### 最近查看

- 使用浏览器 `localStorage` 保存最近查看记录
- 最多保留 10 条
- 相同基金代码会自动去重，保留最近一次查看结果

### 基金分组

- 使用浏览器 `localStorage` 保存分组信息
- 分组名称不可重复
- 单个分组最多保存 10 只基金
- 数据仅保存在当前浏览器本地，不会上传到服务端

## 适配说明

- 桌面端为卡片式双栏布局
- 移动端自动切换为单栏布局
- 不依赖任何前端框架，适合直接嵌入现有 H5 项目或静态页面
