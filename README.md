# 基金实时估值查询

一个使用纯原生 HTML、CSS、JavaScript 编写的基金实时估值查询页面，兼容 PC Web 和 H5 页面展示。

## 功能说明

- 支持输入 6 位基金代码查询实时估值
- 展示基金名称、基金代码、实时净值、前一日净值、净值日期、估值时间
- 根据涨跌幅自动切换展示颜色
- 支持缓存最近查看的 10 条基金记录
- 支持点击历史记录快速再次查询
- 支持清空本地历史记录

## 项目文件

- `index.html`：主页面，包含样式和查询逻辑
- `nginx.conf`：静态部署时可参考的 Nginx 配置

## 接口说明

页面使用东方财富基金估值 JSONP 接口：

`http://fundgz.1234567.com.cn/js/基金代码.js?rt=时间戳`

示例返回：

```js
jsonpgz({
  "fundcode": "017470",
  "name": "嘉实上证科创板芯片ETF发起联接C",
  "jzrq": "2026-04-30",
  "dwjz": "2.5037",
  "gsz": "2.7068",
  "gszzl": "8.11",
  "gztime": "2026-05-06 13:39"
});
```

字段说明：

- `fundcode`：基金代码
- `name`：基金名称
- `jzrq`：净值日期
- `dwjz`：前一交易日净值
- `gsz`：实时估值净值
- `gszzl`：实时涨跌幅百分比
- `gztime`：估值时间

## 使用方式

### 本地打开

直接使用浏览器打开 `index.html` 即可。

### 静态部署

将项目部署到任意静态服务器即可，例如 Nginx。

## 最近查看缓存逻辑

- 使用浏览器 `localStorage` 保存最近查看记录
- 最多保留 10 条
- 相同基金代码会自动去重，保留最近一次查看结果
- 缓存仅保存在当前浏览器本地，不会上传到服务端

## 适配说明

- 桌面端为双栏卡片式布局
- 移动端自动切换为单栏布局
- 不依赖任何前端框架，适合直接嵌入现有 H5 项目或静态页面
